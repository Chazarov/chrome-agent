# Примеры использования Chrome Agent

## Базовые задачи

### 1. Простая навигация
```
Вы: Открой google.com
```

Агент:
- Перейдет на google.com
- Сообщит о результате

### 2. Поиск информации
```
Вы: Найди на Wikipedia статью про Python
```

Агент:
- Перейдет на wikipedia.org
- Найдет поле поиска
- Введет "Python"
- Нажмет на кнопку поиска
- Сообщит о найденной информации

### 3. Заполнение форм
```
Вы: Открой google.com и найди информацию о "artificial intelligence"
```

Агент:
- Откроет Google
- Найдет поле поиска
- Введет текст
- Отправит форму
- Покажет результаты

## Продвинутые задачи

### 4. Многошаговая навигация
```
Вы: Открой github.com, перейди в раздел Explore, и найди популярные Python репозитории
```

Агент выполнит последовательность действий:
1. Навигация на github.com
2. Поиск и клик на Explore
3. Фильтрация по языку Python
4. Анализ результатов

### 5. Извлечение данных
```
Вы: Открой hacker news и расскажи о топ-3 новостях
```

Агент:
- Перейдет на news.ycombinator.com
- Извлечет заголовки и описания
- Отформатирует информацию
- Предоставит краткий обзор

### 6. Взаимодействие с динамическим контентом
```
Вы: Открой reddit.com, прокрути вниз и покажи популярные посты
```

Агент:
- Откроет Reddit
- Выполнит прокрутку
- Дождется загрузки контента
- Извлечет информацию

## Работа с командами

### История сессии
```
Вы: /history
```

Покажет все сообщения текущей сессии с временными метками.

### Новая сессия
```
Вы: /new
```

Сохранит текущую сессию и начнет новую.

### Выход
```
Вы: /exit
```

Корректно завершит работу, сохранив все данные.

## Советы по использованию

### ✅ Хорошие практики

1. **Будьте конкретны**: "Открой google.com и найди информацию о Python" лучше чем "найди что-нибудь"

2. **Разбивайте сложные задачи**: Если задача очень сложная, разбейте ее на подзадачи

3. **Используйте контекст**: Агент помнит предыдущие действия в рамках сессии

### ❌ Чего избегать

1. **Слишком абстрактные задачи**: "Найди что-нибудь интересное" - агент не поймет что делать

2. **Задачи требующие аутентификации**: Агент пока не поддерживает вход в аккаунты

3. **Слишком длинные последовательности**: Более 10 шагов могут привести к ошибкам

## Отладка

### Если агент не понимает задачу
- Переформулируйте более конкретно
- Разбейте на более мелкие шаги
- Используйте /new для новой сессии

### Если страница не загружается
- Проверьте интернет соединение
- Попробуйте другой URL
- Убедитесь что сайт доступен

### Если агент зацикливается
- Используйте Ctrl+C для прерывания
- Агент автоматически остановится через 50 шагов

## Ограничения

Текущая версия не поддерживает:
- ❌ Аутентификацию и логин
- ❌ CAPTCHA
- ❌ Скачивание файлов
- ❌ Загрузку файлов
- ❌ Платежи (безопасность!)
- ❌ Взаимодействие с iframe
