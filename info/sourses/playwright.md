# Playwright 
библиотека для автоматизации работы браузера


# !!! Системные Требования 
- обязательная загрузка браузера через инструменты библиотеки. Команда ```playwright install```. Без выполнения данной команды возникает неочевидная ошибка Критическая ошибка: BrowserType.launch: Executable doesn't exist даже при наличии установленного браузера в системе.
